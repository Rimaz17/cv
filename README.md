# Rimaz Saththar Portfolio

Premium, motion-first developer portfolio built with React, Tailwind CSS, and Framer Motion.

## Tech

- React + Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Run Locally

1. Install dependencies:

	npm install

2. Start development server:

	npm run dev

3. Build production assets:

	npm run build

4. Preview production build:

	npm run preview

## CV Download Setup

Place your CV file in:

public/Rimaz_Saththar_CV.pdf

Both the navbar and hero download buttons are already wired to this exact path.

## Personalization

- Update personal and social links in src/data/portfolioData.js
- Update project links (GitHub and Live) in src/data/portfolioData.js
- Tune visual accents and animations in src/index.css
